This repository contents the deliverables and supporting material 
for the Getting and Cleaning Data course project, specifically:

* run_analysis.R: R script file containing a function of the same
name to process source data into a final summarised set

* CodeBook.md: A description of the data sources used, composition
of the final data set (parameters) and summary of the steps involved
in the data processing

* SummaryDataSet.txt: The final data set to be submitted
